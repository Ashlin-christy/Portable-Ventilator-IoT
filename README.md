# Portable-Ventilator-IoT
This project presents an IoT-Based Portable Ventilator developed to provide a compact, portable, and cost-effective respiratory support solution. The system integrates embedded hardware and IoT technologies to continuously monitor vital respiratory parameters and transmit patient data for remote monitoring. The project demonstrates the application of microcontrollers, sensors, real-time data acquisition, and cloud connectivity in smart healthcare systems.
